## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Get request

```bash
{
  user (id: "62bd5124ce3ef62b36aa25f1"){
    firstName
    lastName
    email
    favouriteArtistIds
  }
}
```

### Post request

```bash
{
  createUser(createUserInput: {
    firstName: "Misha2"
    lastName: "Volk2"
    password: "12345678"
    email: "lev@mail.com"
  })
  {
    lastName
    firstName
    email
    password
    _id
  }
}
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
